## 介绍
Web3 安全是去中心化网络的关键方面。这个项目旨在提供一个全面的指南，帮助理解和掌握 Web3 安全。

## 学习路线
- **智能合约安全**: 探索智能合约中的最佳实践和常见漏洞。
- **区块链协议**: 了解各种区块链协议的安全机制。
- **密码学**: 学习密码学原理及其在 Web3 中的应用。
